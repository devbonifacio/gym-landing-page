## 💻 Sobre o projeto
Landing page moderna de academia desenvolvida com HTML, CSS e JavaScript.

O objetivo do projeto é demonstrar habilidades de front-end criando um site visualmente atrativo, responsivo e com estrutura profissional.

---

## 🚀 Funcionalidades
- Layout moderno e responsivo 📱
- Seção de hero impactante
- Cards de benefícios
- Planos de assinatura
- Seção de resultados
- Formulário de contato com interação em JavaScript

---

## 🛠️ Tecnologias utilizadas
- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (Vanilla)
